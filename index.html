HTML: Building a the foundation of your house

Tags

  • Code is like a House
  • HTML (Hyper Text Markup Language) uses tags to break up our content into different elements the same way walls divide a house into different rooms
  • Example: the <h1>, defines you're largest heading 
    • 
    <h1>Bon Jour World</h1>
  • Elements are defined by opening and closing tags
  • There are numbered headings from h1 to h6
  • Without tags, our page would just be a wall of text
Some Key Tags:
<p> paragraph, define you're regular text
<ul>, <li> make a list
<a> anchor create links

You Do:

In the file you downloaded, (click here if you haven't already) you'll see a note that says "add header here" try adding a header and a pargraph underneath it. If you're feeling creative use the <strong> tag to bold a few of the words.

Tables
  • The whole structure of a table and of HTML in general is elements nested inside each other 
        
    <p>
        <span>
        </span>
    </p>
  • Emails use the <table>, <tr> and <td> tags to create rows and columns for our content to go in
    •     
    <table>
        <tbody>
            <tr>
                <td>
                </td>
            </tr>
        </tbody>
    </table>
Table Tags
<table> container for the table
<tbody> container for the tables contents
<tr> defines a row
<td> defines a column within a row

(Web pages use a different kind of tags called a "div" tag to divide the page into sections, but today we'll just focus on emails)

You Do:

In your file, there's another note asking you to add some columns. Create two columns here and add a paragraph in each.

Images

  • Inside the image tag we put a source attribute
  • Unlike almost every other element, image's don't need a closing tag
  • To make our image fit to our cell give it a width of 100%
    • 
    <img width="100%"url="http://placekitten.com/">
  • File size increases the likelihood of your email being sent to spam, so let's keep it to a minimum
  • With emails all image tag src attributes must lead to an online image, not to your images on your computer

You Do:

In your file, add this image to your email, where it says "add image" and give it a width of 100%

CSS: Decorating our House

  • If HTML is the walls of the house, CSS (Cascading Style Sheets) is the paint, wallpaper and carpeting
  • CSS controls the colors, sizes and fonts, on our site among other things
  • Again, this works differently on the web, but in emails we write our CSS styles within the HTML tags like this
    •     
    <p style="color: green; font-weight: bold; font-family: sans-serif">
    </p>
  • Here are some fun CSS properties you can play with

You Do:

Go through you're demo file and change the colors of our h1 headers to blue and the background color of our bottom row to blue.

Positioning with CSS
  • To add space around the contents of your cells increase the cell padding in your table tag, like so:
    •     
        <table cellspacing="0" cellpadding="10" border="0">
  • You can also set the width of your table, but Yahoo and Gmail will have a pre-made body element with its own width that your email will go inside of, so your table's width should never be more than 98%
  • Adding "margin: 0 auto" your outer table will center it in on the page

Note: Styles like "padding" and "margins" can be very useful, but avoid adding them to your content, since some email clients like gmail will strip them out

Web V. Email: Cheat Sheet

Website:

  • Always use classes to do your styling
  • Write your CSS in an external page and link to it from your HTML
  • Save out the images you are using in the same folder as your html doc, the you got them from may take them down
  • Use divs and sections to break up your page
  • Use margins and padding and to position things
  • Always include , html tags, a head section and a body section; (this website has folders you can download with the correct file structure for web all set up.)

Email:

  • Do all your styles inline
  • Use classes very sparingly
  • If you have to use Classes, include them in a style tag at the head of your email (YOU CANNOT LINK TO ANYTHING OUTSIDE YOUR HTML FILE)
  • Only link to images online, for the same reason, you can only use the code from your HTML file
  • Use tables within tables to position or you can use padding- donâ€™t add margins to the content within your tables
  • You don't need to include a header, some email clients will strip it out

* A good rule of thumb is be ok with the way your email looks sans-classes
